<h1 align="center">Inventory System (C++)</h1>

<p align="center">
A simple and efficient inventory management system built using C++.
</p>

<hr>

<h2>📌 Overview</h2>
<p>
This project demonstrates a basic inventory system where users can:
</p>
<ul>
  <li>Add items to inventory</li>
  <li>Sell items</li>
  <li>Automatically remove items when quantity reaches zero</li>
  <li>View all available items</li>
</ul>

<p>
The system was improved from a basic implementation by:
</p>
<ul>
  <li>Fixing item removal logic</li>
  <li>Eliminating memory leaks</li>
  <li>Using better data handling practices</li>
</ul>

<hr>

<h2>⚙️ Tech Stack</h2>
<ul>
  <li>C++ (STL basics)</li>
  <li>Object-Oriented Programming (OOP)</li>
</ul>
